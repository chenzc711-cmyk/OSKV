# 天猫店铺仓库管理系统

一个零依赖、可直接运行的前端仓库管理系统，适合天猫店铺进行商品资料、成本、库存和出入库记录管理。

## 最简单使用方式（推荐）

如果你不熟悉命令行，直接使用单文件版本：

1. 下载或复制 `天猫仓库管理系统.html` 到电脑桌面。
2. 双击 `天猫仓库管理系统.html`。
3. 浏览器打开后即可添加商品、导入库存、做入库/出库记录。

> 这个单文件已经包含页面、样式和功能代码，不需要 Bash、PowerShell、Python，也不需要安装任何依赖。

## 功能

- 数据仪表盘：展示 SKU 数量、在库总数、库存成本和打包成本。
- 手动添加商品：维护商品名称、商品规格、商品成本、打包成本和在库数量。
- 批量导入商品：支持 CSV 文件上传，也支持粘贴 CSV 文本导入。
- 库存操作：可按商品执行入库、出库、删除和搜索。
- 出入库记录：自动记录新增、入库和出库流水。
- 本地存储：使用浏览器 localStorage 保存数据，无需后端服务。

## 其他运行方式

如果你熟悉命令行，也可以使用拆分文件版本，在项目根目录运行：

```bash
python3 -m http.server 4173
```

然后访问 <http://localhost:4173>。

也可以直接打开 `index.html`，但最方便给非技术用户使用的是 `天猫仓库管理系统.html`。

## CSV 模板

```csv
商品名称,商品规格,商品成本,打包成本,在库数量
示例商品,标准款,10.00,1.00,100
```
