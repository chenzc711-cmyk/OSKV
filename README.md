# 仓库管理系统

这是一个仿参考图制作的仓库管理系统，包含左侧导航、仪表盘、库存查询、入库登记、出库登记、成本登记、商品管理和报表统计页面。

## 最简单使用方式

下载或复制单文件版本到电脑桌面：

```text
天猫仓库管理系统.html
```

然后直接双击打开即可使用，不需要 Bash、PowerShell、Python，也不需要安装依赖。数据会保存在当前浏览器本地。

## 多电脑同步使用说明

当前版本是纯前端本地版，数据保存在每台电脑自己的浏览器里，因此不同电脑不会自动同步。如果需要多台电脑同步，有三种做法：

1. **临时共享**：在一台电脑导出 CSV，再到另一台电脑导入。
2. **局域网共享**：把系统放在一台固定电脑或 NAS 上，其他电脑通过同一个网址访问；但仍建议后续接数据库保存数据。
3. **正式多人同步**：增加后端、数据库和账号权限，把商品、入库、出库、成本数据统一保存到服务器。

## 分文件运行方式

如果你下载的是完整项目，也可以在项目目录运行：

```bash
python3 -m http.server 4173
```

然后访问：

```text
http://localhost:4173
```

## 功能

- 仪表盘：商品总数、总库存、库存总额、库存预警、今日/本月出入库、TOP 图表。
- 库存查询：搜索、库存状态筛选、库存预警、导出 CSV。
- 入库登记：新增入库、批量导入、导出 CSV、日期筛选。
- 出库登记：新增出库、批量导入、导出 CSV、日期筛选。
- 成本登记：新增成本、成本类型筛选、导出 CSV。
- 商品管理：新增、编辑、删除商品，搜索和导出。
- 报表统计：出入库统计、成本汇总、趋势图展示。

## 批量导入格式

入库/出库批量导入 CSV 可使用以下格式：

```csv
商品名称,数量,日期
腰靠-米白色,10,2026-05-28
汽车头枕-米白色,5,2026-05-28
```
