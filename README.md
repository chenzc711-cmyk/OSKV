# 拾纷电商汽车用品仓库管理系统

这是一个按参考界面重新设计的汽车用品仓库管理系统，包含仪表盘、库存查询、入库登记、出库登记、退货入库、成本登记、商品管理和利润计算。系统为纯前端版本，数据保存在当前浏览器本地，不需要安装后端或数据库。

## 最简单使用方式

下载或复制单文件版本到电脑桌面：

```text
新版仓库管理系统.html
```

然后直接双击打开即可使用。它已经包含页面、样式和功能代码，不需要 Bash、PowerShell、Python，也不需要安装依赖。

手机也可以打开：把 `新版仓库管理系统.html` 发到手机，或把文件放到同一局域网电脑/NAS 上用手机浏览器访问。页面已经做了手机端适配，会自动变成顶部横向导航、卡片单列和可滑动表格。

> 也可以打开 `天猫仓库管理系统.html`，两个单文件版本内容一致。

## 如何确认自己打开的是最新版

最新版页面左上角只显示品牌名：

```text
拾纷电商
```

仪表盘大标题为：

```text
拾纷电商汽车用品仓库管理系统
```

如果还看到“参考图新版”“星云仓储中枢”“天猫店铺”或旧的红色按钮界面，说明你打开的是旧文件或浏览器缓存。请重新从 GitHub 下载最新 ZIP，或打开新版 HTML 后按 `Ctrl + F5` 强制刷新。

## 从 GitHub 下载 ZIP

1. 回到仓库首页的 **代码 / Code** 页面。
2. 点击绿色 **Code** 按钮。
3. 选择 **Download ZIP**。
4. 解压 ZIP。
5. 双击 `新版仓库管理系统.html`。

不要在 GitHub 冲突页面下载或打开 `.zip` 文件；GitHub 网页冲突编辑器不支持处理二进制 ZIP。

## 本次优化内容

- 品牌统一为“拾纷电商”，去掉侧边栏小字和“参考图新版”标识。
- 仪表盘标题改为“拾纷电商汽车用品仓库管理系统”，保留更新时间但去掉旧版本文字。
- 商品分类统一叫“规格分类”，库存、入库、出库、商品管理均使用同一套规格分类筛选。
- 入库登记支持日期筛选、规格分类筛选、全选勾选、备注记录和编辑入库记录，修改后会自动校正对应商品库存。
- 库存查询按商品基础名称和规格分类排序，同一商品的不同规格会排在一起。
- 商品管理增加“批量添加商品”，支持用 CSV/XLSX 一次导入多个商品；导入时如果规格分类不是系统已有分类，该商品会自动跳过，避免产生错误分类。
- 添加“退货入库”模块，可登记并编辑退货日期、退款创建时间、订单号、物流公司、物流单号、物品名称、物品数量、操作人和备注，并自动回补库存。
- 添加出库/入库弹窗中的商品下拉按商品基础名称分组排序，同一商品的不同规格更容易连续选择。
- 取消原“报表统计”入口，新增“利润计算”界面，按销售收入、入库采购成本、费用成本和商品毛利估算利润。
- 仪表盘取消库存价值/出库数量/出库金额 TOP5 卡片，将渠道款项移到下方，并新增普通渠道商、3W品牌渠道、台州伊文渠道三个余额，系统会自动合并计算“渠道剩余款项合计”。
- 增加手机网页端适配：手机打开时左侧菜单会变成顶部横向导航，表格可左右滑动，弹窗和按钮更适合触屏操作。
- 入库登记支持编辑入库单价，每条入库记录都可单独修改采购单价，利润计算会按入库单价 × 数量计算采购成本。
- 利润计算升级为详细登记区，销售收入、入库采购成本、费用成本都可手动登记、编辑、删除和导出。

## 功能说明

- **仪表盘**：查看商品总数、总库存、库存总额、库存预警、今日/本月出入库，以及普通渠道商、3W品牌渠道、台州伊文渠道和渠道剩余款项合计。
- **库存查询**：按商品名称、编码、规格分类搜索；按库存状态和规格分类筛选；支持导出 CSV。
- **入库登记**：新增入库、编辑入库、修改入库单价、填写备注、CSV/XLSX 批量导入、日期筛选、规格分类筛选、全选和删除记录。
- **出库登记**：新增出库、CSV/XLSX 批量导入、日期筛选、规格分类筛选和删除记录。
- **退货入库**：登记/编辑退货日期、退款创建时间、订单号、物流公司、物流单号、退回物品、数量、操作人和备注，保存后自动增加对应商品库存，并支持 CSV/XLSX 批量导入。
- **成本登记**：登记采购、快递、代发和其他成本，支持类型筛选和导出。
- **商品管理**：新增、编辑、删除、批量添加商品，并统一维护规格分类；批量导入只接受已有规格分类。
- **利润计算**：汇总销售收入、入库采购成本、费用成本和预估利润，并按商品展示出库收入、入库成本、退货数量和预估毛利；支持登记/编辑/删除收入、采购成本和费用调整。

## CSV/XLSX 批量导入格式

### 商品批量添加

> 注意：`规格分类` 必须是系统里已经存在的分类，例如汽车头枕、腰靠、护肩套、坐垫、方向盘套、其他；如果 CSV/XLSX 里出现不存在的分类，该行商品不会导入。

```csv
商品名称,规格分类,单位,单价,当前库存,预警库存
汽车头枕,雅黑色,只,23.90,30,5
汽车头枕,米白色,只,23.90,26,5
腰靠,雅黑色,个,24.90,16,5
```

### 入库/出库批量导入

CSV 和 XLSX 都按同样的列顺序填写。入库时第 4 列会作为供应商、第 5 列作为备注、第 6 列可填写入库单价；出库时第 4 列会作为用途。

```csv
商品名称,数量,日期,供应商或用途,备注,入库单价
汽车头枕-米白色,5,2026-05-28,补货供应商A,新品补货,21.50
腰靠-雅黑色,10,2026-05-28,销售出库,直播订单,
```

### 退货入库批量导入

```csv
订单号,物流单号,物流公司,商品名称,数量,操作人,备注,退货日期,退款创建时间
TM20260528001,SF1234567890,顺丰速运,汽车头枕-米白色,1,陈,客户退回,2026-05-28,2026-05-27
```

## 多电脑同步使用

当前版本数据保存在每台电脑自己的浏览器里，不会自动同步。多电脑使用有三种方式：

1. **手动同步**：一台电脑导出 CSV，另一台电脑导入。
2. **局域网共享页面**：把系统文件放在一台固定电脑或 NAS 上，其他电脑访问同一个页面；但数据仍建议定期导出备份。
3. **正式多人版**：后续增加服务器、数据库和账号权限，所有电脑共用同一份数据库。

## GitHub 冲突处理提示

如果 PR 页面提示冲突，建议不要在旧 PR 上反复合并。最稳妥的方法是：

1. 关闭旧的冲突 PR。
2. 回到最新 `main` 分支。
3. 新建一个分支，重新提交新版文件。
4. 再开新的 PR。

如果必须在网页解决冲突，请保留包含“拾纷电商汽车用品仓库管理系统”“tmallWarehouseV10”“新版仓库管理系统.html”的内容，并删除冲突标记 `<<<<<<<`、`=======`、`>>>>>>>`。

## 微信小程序版本

已经新增 `miniprogram/` 目录，可用微信开发者工具打开：

1. 安装并打开 **微信开发者工具**。
2. 选择 **导入项目**。
